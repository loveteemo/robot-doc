# 收到文本消息

> 收到文本消息

## 请求参数

| 参数       | 类型     | 描述      | 
|----------|--------|---------|
| content	 | String | 内容      |
| id       | String | 	时间戳    |
| id1      | String | 	根据场景变换 |
| id2      | String | 	根据场景变换 |
| id3      | String | 根据场景变换  |
| srvid    | String | 	服务ID   |
| time     | String | 	昵称     |
| type     | String | 	1      |
| wxid     | String | 	发生微信ID |

## 示例

```json
{
  "content":"hello",
  "id":"20230711154020",
  "id1":"",
  "id2":"",
  "id3":"",
  "srvid":1,
  "time":"2023-07-11 15:40:20",
  "type":1,
  "wxid":"wxid_1"
}
```
