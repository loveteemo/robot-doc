# 被动操作

当需要获取微信接收到的消息后，进行处理。需要使用到被动操作。

也可以理解为了启动ws之后，收到的消息的主动推送到执行地址。用于处理接受消息的回调函数。

这样也可以实现自动回复功能。可以用任何语言来实现我们需要的功能。

例如：接收到 "签到" 后，自动回复 "签到完成"
